Documentation

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.










Request HTTP Method Type URL Response Data Example
Return a list of All Movies to the user GET /movies A JSON object holding data about all the movies { "Actors": ["String"],
"_id": "String",
"Title": "String",
"Description": "String",
"ImagePath": "String",
"Featured": "Boolean"
},
Return data about a single movie by title to the user GET /movies/:Title A JSON object holding data about a single movie { "Actors": ["String"],
"_id": "String",
"Title": "String",
"Description": "String",
"ImagePath": "String",
"Featured": "Boolean"
}
Return data about a genre (description) by name/title GET /movies/genre/:Name A JSON object holding data about the movie genre {
"_id" : "Integer",
"Genre Name" : "String",
"Description" : "String"
}
Return data about a director (bio, birth year, death year) by name GET /movies/directors/:Name A JSON object holding data about the director {
"_id" : "Integer",
"Director Name" : "String",
"Bio" : "String",
"Birth" : "Date",
"Death" : "Date"
}
Allow new users to register POST /users A JSON object holding data about the new user, including an ID {
"_id" : "Integer",
"Username" : "String",
"Password" : "String",
"Email" : "String",
"Birthday" : "Date",
"FavoriteMovies" : [ "String" ]
}
Allow users to updated their user info PUT /users/:id A JSON object holding updated data of the user {
"_id" : "Integer",
"Username" : "Updated String",
"Password" : "Updated String",
"Email" : "Updated String",
"Birthday" : "Updated Date",
"FavoriteMovies" : [ "Updated String"]
}
Allow users to add a movie to their list POST /users/id/:movieTitle A JSON object holding updated data on their list of movies {
"_id" : "Integer",
"Username" : "String",
"Password" : "String",
"Email" : "String",
"Birthday" : "Date",
"FavoriteMovies" : [ "Updated String", "Updated String"]
}
Allow users to remove a movie from their list DELETE users/:id/:movieTitle A JSON object holding updated data on their list of movies {
"_id" : "Integer",
"Username" : "String",
"Password" : "String",
"Email" : "String",
"Birthday" : "Date",
"FavoriteMovies" : [ "Updated String"]
}
Allow existing users to deregister DELETE /users/:id A string stating that the user has been removed "User" has been removed.
